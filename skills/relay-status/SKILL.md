name: relay-status
description: Show relay connection status, session ID, and check for unread messages from paired sessions.
---

Check the relay connection status and read any incoming messages.

Steps:
1. Call the `relay_status` tool to show the current session ID and connection state
2. Call the `relay_read` tool to fetch and display any unread messages from the peer
3. Present the status and messages clearly to the user
